Subir a GitHub

git init
hacer el add
hacer el commit
subir todo a la rama principal con git branch -M master
poner la ruta con git remote add origin (la ruta)
setear la url con git remote set-url origin https://github.com/nachostylescoc/NOMBRE-DEL-REPO.git
hacer el push con git push -u origin master

con esto ya se creo el seteo y para subir nuevos archivos basta con hacer git push


HACER PUSH POR PRIMERA VEZ EN UNA PC NUEVA

por HTTPS

Al hacer git push -u origin master nos pedirá usuario y contraseña, la contraseña será un token que hay que generar en las settings de GitHub. Sin embargo si lo hacemos asi tal cual siempre que abramos una nueva terminal nos pedirá el token, para dejar el permiso para siempre usamos:

git config --global credential.helper store

hacemos git push, introducimos usuario y token y el permiso quedaría para siempre
